##Equity Positions Manager
A simple React app to manage equity positions based on trade transactions with support for INSERT, UPDATE, 
and CANCEL actions. Users can submit trades through a form and view current security positions in real-time.

#Features

Submit trades with all necessary fields (Transaction ID, Trade ID, Version, etc.)

Automatically calculate and maintain equity positions

Supports trade versioning with INSERT, UPDATE, CANCEL logic

Tab-based UI for trade submission and position view

Styled with clean, responsive CSS

Built-in testability and modular structure

## Run
npm start
# or
yarn start
Navigate to http://localhost:3000 in your browser.

##Tests

npm test
# or
yarn test
